# Placement Cell Website

A professional, institutional-grade website for university/college placement cells.

## Features

- **Institutional Design**: Formal, trustworthy, data-centric interface
- **Theme Toggle**: Light/Dark mode with localStorage persistence
- **Responsive**: Works perfectly on mobile, tablet, and desktop
- **8 Pages**: Complete navigation structure
- **Statistics Dashboard**: Chart.js integration for data visualization
- **Clean Architecture**: Organized file structure

## Design Philosophy

- **Clarity > Visuals**: Institutional credibility over flashy effects
- **Data-centric**: Statistics and information first
- **Formal Tone**: University placement portal aesthetic
- **Limited Effects**: Subtle glassmorphism and neumorphism only where appropriate

## Technologies

- HTML5
- Tailwind CSS
- Chart.js (for statistics)
- Vanilla JavaScript

## Usage

1. Open `index.html` in a web browser
2. Navigate through pages using the navbar
3. Toggle theme using the button in the navbar
4. View statistics and data visualizations

## Browser Support

Modern browsers with CSS Grid and Flexbox support.
